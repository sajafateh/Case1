# Builder Design Pattern – Game Character Case Study

## Project Overview
This project demonstrates the use of the Builder Design Pattern in Java through a case study of creating a game character.  
The objective is to illustrate how complex objects can be constructed step by step while improving code readability, flexibility, and maintainability.

## Design Pattern Used
- Builder Design Pattern (Creational Pattern)

## Case Study: Game Character Creation
In this project, a game character is created with multiple attributes such as:
- Name  
- Type  
- Health  
- Weapon  
- Armour  
- Skills  

The Builder pattern is applied to construct the character incrementally, avoiding common issues such as:
- Telescoping constructors  
- Too many parameters  
- Inconsistent object state
